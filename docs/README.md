## 새 문서 카테고리를 추가할 때
1. `docs/<카테고리명>/` 디렉터리 생성.
2. 그 안에 `README.md` (사람용 인덱스) 와 `AGENTS.md` (AI 규칙) 를 함께 둔다.
3. 위 표에 한 줄 추가.

향후 추가 후보: `runbooks/`, `api/`, `user-guide/`, `onboarding/`, `architecture/`.
